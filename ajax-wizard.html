Ajax Wizard
Build an Ajax Function mostly comes with a difficult workaround and sometimes it sucks a lot until it runs.

This circumstances inpired me to code a lightweight Ajax API. The benefits of Ajax Wizard is, you can write your parameters right into your HTML element, so there is no code change on further extensions.

All you need to run Ajax Wizard is jQuery and a running PHP environment. Checkout this simple Ajax Demo by clicking the demo button below.

Setup
To install Ajax Wizard, unpack the zip file in your web directory and add the following line inside the head section off your HTML document.

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.1.1/jquery.min.js">
<script src="/ajax-wizard/src/controller.js">
